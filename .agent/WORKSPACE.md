# Vyasa Workspace Map

This document outlines the purpose of each repository in the Vyasa project workspace and their logical names for reference.

| Repository | Logical Name | Purpose |
| :--- | :--- | :--- |
| `project-vyasa/vyasa` | **Compiler** | Core Rust compiler (`vyasac`) and language semantics. |
| `project-vyasa/vyasa-docs` | **Docs** | Documentation site (`astro-starlight`) and sample workspaces. |
| `project-vyasa/vedabase.io`| **Vedabase** | Scripts and tools for processing Vedabase content. |
| `project-vyasa/vedic-text` | **Vedic-Text** | Raw text data and content corpus. |
